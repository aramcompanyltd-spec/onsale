퓨전 아케이드 소개

퓨전 아케이드는 회원가입, 설치, 결제 없이 누구나 브라우저에서 바로 즐길 수 있는 무료 게임 사이트입니다. 클래식 아케이드 장르(스네이크, 퐁, 벽돌깨기 등)에서 영감을 받아 모든 게임을 직접 제작했으며, 특정 게임기의 롬 파일이나 타사 저작물을 사용하지 않습니다.

// 우리의 원칙

1. 회원가입이나 개인정보 제공 없이 누구나 즉시 플레이할 수 있어야 합니다.
2. 모든 게임 콘텐츠는 자체 제작하거나 명확한 라이선스가 확인된 것만 사용합니다.
3. 광고는 게임 플레이를 방해하지 않는 선에서만 배치합니다.

// 기술

모든 게임은 순수 HTML5 캔버스와 자바스크립트로 제작되었으며, 별도의 플러그인이나 앱 설치 없이 최신 브라우저에서 바로 실행됩니다. 최고 점수와 같은 개인 기록은 사용자의 브라우저(로컬 스토리지)에만 저장되며 서버로 전송되지 않습니다.

// 문의
