벽돌깨기 (BREAKOUT)

SCORE 0 LIVES 3

// ABOUT THIS GAME

벽돌깨기는 패들로 공을 튕겨 화면 위쪽에 배치된 벽돌을 모두 부수는 클래식 반사신경 게임입니다. 공을 놓치면 목숨이 하나씩 줄어들며, 목숨을 모두 잃으면 게임이 종료됩니다. 모든 벽돌을 부수면 클리어입니다.

마우스나 터치, 키보드로 패들을 조작할 수 있으며 별도의 설치나 로그인 없이 바로 플레이할 수 있는 오리지널 구현입니다.

// HOW TO PLAY

  • 마우스를 좌우로 움직여 패들 조작
  • 또는 ←→ 방향키 사용
  • 공을 놓치면 목숨 -1, 목숨 3개 소진 시 종료
  • 모든 벽돌을 부수면 클리어
