스네이크 (SNAKE)

SCORE 0 BEST 0

// ABOUT THIS GAME

스네이크는 화면 속 뱀을 조작해 먹이를 먹을 때마다 몸통이 길어지는 클래식 아케이드 게임입니다. 벽이나 자기 자신의 몸에 부딪히면 게임이 끝나며, 몸통이 길어질수록 공간 관리와 순간적인 판단력이 더 중요해집니다. 이 페이지의 스네이크는 별도의 설치나 로그인 없이 브라우저에서 즉시 실행되는 오리지널 구현입니다.

최고 점수는 이 브라우저에 자동으로 저장되어, 다시 방문했을 때 이전 기록과 비교하며 도전할 수 있습니다. 별도의 계정이나 서버 저장은 사용하지 않으므로 개인정보 입력이 전혀 필요 없습니다.

// HOW TO PLAY

  • ↑↓←→ 방향키로 이동
  • 노란색 먹이를 먹으면 점수 +10
  • 먹을수록 속도가 빨라집니다
  • 벽 또는 자기 몸통에 닿으면 종료
  • 모바일은 화면 하단 버튼 사용
