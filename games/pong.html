퐁 (PONG)

YOU 0 CPU 0

// ABOUT THIS GAME

퐁은 1970년대 초기 아케이드 게임에서 영감을 받은 2인 탁구형 반사신경 게임입니다. 왼쪽 패들은 플레이어가, 오른쪽 패들은 컴퓨터가 조작하며, 공을 놓치는 쪽이 실점합니다. 먼저 7점을 획득하면 승리합니다.

이 페이지의 퐁은 마우스, 터치, 키보드를 모두 지원하는 오리지널 구현으로, 어떤 기기에서도 별도 설치 없이 즉시 플레이할 수 있습니다.

// HOW TO PLAY

  • 마우스를 위아래로 움직여 패들 조작
  • 또는 ↑↓ 방향키 사용
  • 공을 놓치면 상대가 득점
  • 먼저 7점을 내면 승리
